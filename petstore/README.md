## Test REST API

Build Souce Code, generate `petstore.war`, deploy it to WildFly, then test the following API:

* http://localhost:8080/petstore/rest/helloworld/json 
* http://localhost:8080/petstore/rest/helloworld/xml
* http://localhost:8080/petstore/rest/pet/1
* http://localhost:8080/petstore/rest/store/order/1
* http://localhost:8080/petstore/rest/user/user1
